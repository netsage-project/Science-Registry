Netsage Science Registry mmdb file hosting

You can download the mmdb files as follows: 

wget https://netsage-project.github.io/Science-Registry/scireg.mmdb
wget https://netsage-project.github.io/Science-Registry/communities.mmdb

To access the JSON files used to generate these, go to:
    https://netsage-project.github.io/Science-Registry/scireg.json
    https://netsage-project.github.io/Science-Registry/communities.json

To access python scripts used to generate the JSON and mmdb file, go to:
    https://github.com/netsage-project/netsage-pipeline/tree/master/scireg-utils
